# Unix Shell

~ - замества home  
. - настояща директория  
.. - една директория нагоре  


## Файлови Команди  
---------------------------------------  

**_ls_** - листва директория  
**_ls -la_** - форматирано листва директория и скритите файлове  
**_cd dir_** - отива в директория dir  
cd - отива в директория home  
pwd - показва настоящата директория
